# Product Guide Images

## Required Images

These images should be exported from Figma node 1:4251 or created as 3D renders matching the design.

### 1. sleeper-sofa.png

- **Dimensions:** 800x600px (4:3 aspect ratio)
- **Description:** 3D render of sleeper sofa support system with golden/yellow cushion
- **Background:** Transparent or light gray (#F6F6F6)
- **Perspective:** Isometric view, slight angle showing depth

### 2. sofa-chair.png

- **Dimensions:** 800x600px (4:3 aspect ratio)
- **Description:** 3D render of sofa/chair support system with orange cushion
- **Background:** Transparent or light gray (#F6F6F6)
- **Perspective:** Isometric view, slight angle showing depth

### 3. bed-support.png

- **Dimensions:** 800x600px (4:3 aspect ratio)
- **Description:** 3D render of bed support system with burgundy/maroon cushion
- **Background:** Transparent or light gray (#F6F6F6)
- **Perspective:** Isometric view, slight angle showing depth

## Export Settings (from Figma)

- Format: PNG with transparency
- Scale: 2x for retina displays
- Color profile: sRGB
- Compression: Optimized for web
